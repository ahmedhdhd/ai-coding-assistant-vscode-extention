
# AI Coding Assistant Extentionsion for VS Code

A lightweight VS Code extension that adds an AI assistant directly inside your editor.
Chat with models, get code suggestions, apply changes instantly, and use custom context to guide the AI.

## Features

* **AI Chat Panel** – Talk to any supported AI model
* **Code Suggestions** – Accept, decline, or apply code directly into your editor
* **Custom Context** – Add project rules or preferences for better responses
* **Multiple Models** – Supports different providers (Groq, Google, etc.)
* **Agent Support** – Automate tasks and file actions inside your workspace
* **Editor Integration** – Insert or replace code in one click

## Setup

1. Get an API key from your AI provider (Groq, Google, etc.)
2. Open VS Code → **Settings**
3. Search for **AI Assistant**
4. Add your API key and select your model

## Usage

* Open the assistant from the activity bar or Command Palette
* Chat with the model and receive code responses
* Manage context rules to guide the assistant
* Apply code suggestions directly to your editor



## Development

```bash
npm install
npm run compile
npm run watch
```

Press **F5** to launch the extension in development mode.
